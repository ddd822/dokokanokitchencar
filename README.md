# キッチンカーコミュ

## サイト概要
### サイトテーマ
好きなキッチンカーを紹介したり、どこにどんなキッチンカーが出店しているか検索できるコミュニティサイト
​
### テーマを選んだ理由
キッチンカーに興味があり、開業について少し調べたことがあるのですが、その中でお客さんに見つけてもらいづらいという側面があるなあと感じていました。  
お客さん側が良いなと思ったキッチンカーを紹介したり、お店側が自分たちの活動を紹介したりできる場所があれば便利ではないかと考えこのテーマにしました。
​
### ターゲットユーザー
- 近くで飲食店を探していて、いつもと違う食事をしたい人
- 好きなキッチンカーを誰かに紹介したい人
- キッチンカーを経営していて宣伝をしたい人
​
### 主な利用シーン
- 近くの飲食店を検索したい時
- 好きなキッチンカーを紹介する時
- ​自分が経営するキッチンカーを宣伝したい時
